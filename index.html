Memuat jadwal dari Google Calendar...
Cara Kelola Jadwal
  1. Login ke Google Calendar dengan workspacerivero@gmail.com
  2. Buat event baru: Nama Ruangan — Nama Klien
  3. Set tanggal, jam mulai, dan jam selesai
  4. Pastikan kalender diset Public agar embed bisa tampil
  5. Website public otomatis update dalam beberapa menit ✅
🔒
Keamanan: Kalender publik Rivero diakses via embed resmi Google (iFrame). Tidak ada API key atau backend yang terekspos. Data booking sensitif tetap aman di Supabase.
